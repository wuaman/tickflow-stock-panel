# 发布说明

## v0.1.60

### ✨ 新增

- **实时行情分档**：明确 None / Free / Starter+ 三档边界。
  - None：仅历史日K（当日数据盘后 1-2 小时可用），无法开启实时行情。
  - Free：自动监控「自选」页面前 5 个标的，最低 6 秒刷新。
  - Starter+：使用全市场实时行情。
- **自选股「移到顶部」**：自选页代码名称列新增移顶按钮，可调整实时监控的前 5 个优先级。
- **复盘页（Beta）**：新增市场复盘入口，左侧菜单标注 Beta。
- **数据范围配置**：盘后管道支持页面化配置拉取范围与扩展数据提供方。

### 🛠 优化

- 监控设置页：Free 档自选股实时卡片改为高亮提示，并展示「代码 + 名称」。
- 档位说明：API Key 检测问号弹层补充「高等档位包含较低档位全部权益」提示，更新 Free / Starter / Pro 档位描述。
- 左侧实时行情入口图标改为设置齿轮；None 档补充注册 TickFlow 链接。
- README 同步：档位边界、None/Free 数据通道、自选实时监控说明统一。

### 🐛 修复

- 修复盘后管道 `_prefs` 变量作用域导致的同步崩溃。
- 修复调整轮询间隔滑动条时的请求风暴崩溃（改为 2 秒防抖保存）。
- 修复自选股实时卡片只显示代码不显示名称的问题。
